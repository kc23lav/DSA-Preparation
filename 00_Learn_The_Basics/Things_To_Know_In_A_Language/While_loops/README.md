## While Loops

This section contains programs based on `While` loop iteration.

### Concepts Covered
- while loop syntax
- entry-controlled loop
- loop conition and termination
- use of break statement

### Problems Implemented
- Print numbers from 1 to N
- Sum of digits if a number
- Reverse a number
- Palindrome number check
- Factotial of a number


### Languages Used
- Java
- C++

### Goal
- Understand condition-based iteration
- Build logic for digit manipulation
- Handle cases where iterations are not fixed
- Prepare for loops-based interview questions
