## For Loops

This section contains programs based on `for` loop iteration.

### Concepts Covered
- for loop syntax
- loop initialization, condition, increment/decrement
- nested for loops
- counting and accumulation logic

### Problems Implemented
- Print numbers from 1 to N
- Sum of first N natural numbers
- Multiplication table
- Factorial of a number
- Count digits in a number

### Languages Used
- Java
- C++

### Goal
- Build strong iteration logic
- Understand repetitive execution
- Prepare for loops-based interview questions
