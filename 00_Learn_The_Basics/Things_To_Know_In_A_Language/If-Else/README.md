## If – Else

This folder contains beginner-level programs based on conditional statements.

### Concepts Covered
- if statement
- if–else
- else-if ladder
- Nested if–else

### Languages Used
- Java
- C++

### Problems Implemented
- Even or Odd number
- Positive / Negative / Zero
- Largest of two numbers
- Largest of three numbers
- Voting eligibility
- Leap year check

### Goal
- Build logical thinking
- Understand decision-making in programs
