## Basic Maths

This section contains important number-based problems used to build
strong mathematical and logical foundations for DSA.

### Problems 
- Count all digits of a number
- Reverse a number
- Check palindrome number
- GCD of two numbers
- Check Armstrong number
- Print all divisors of a number
- Check if a number is prime

### Languages Used
- Java
- C++

### Goal
- Strengthen basic mathematical reasoning
- Understand number manipulation using loops
- Prepare for common number-based interview problems
